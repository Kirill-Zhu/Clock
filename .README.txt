чтобы утановить будильник нажимте иконку "Часов" в правом верхнем углу> 
чтобы установить числовые значения нажмите на кнопку с числовым значением часов или минут и введите нужное число
чтобы установить будильник через стрелку выберите АM или PM (кнопка слева сверху) и тащите стрелку в нужное место
чтобы завести будильник нажмите галочку внизу справа(когда будильник взведен будет зеленая иконка по цетру и красная если -нет)
чтобы выключить будильник нажмите крестик внизу слева
